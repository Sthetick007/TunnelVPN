# HexTunnel VPN Panel

A lightweight SSH/VPN panel for Ubuntu servers.

## Requirements
- Ubuntu 20.04 / 22.04
- Root access
- Fresh VPS recommended

## One Line Install
```bash
bash <(curl -Ls https://raw.githubusercontent.com/Sthetick007/TunnelVPN/main/install.sh)
```

## What Gets Installed
- SSH Menu
- Vmess Menu
- Trojan Menu
- Vless Menu
- System Menu
- Status Service monitor
- SSH Banner

## Supported Protocols
- OpenSSH
- Dropbear
- WebSocket SSH
- Vmess (Xray)
- Trojan (Xray)
- Vless (Xray)
- SOCKS5

## After Install
Type `menu` in terminal to open the panel.
